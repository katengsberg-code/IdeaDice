# IdeaDice
A shared decision-making app for couples, friends, and families.

DESCRIPTION:
IdeaDice is designed to record and recommend shared ideas for activities
among couples, friends, and families, and recommend activities to reduce
decision fatigue.

Users can store restaurants, movies, recipes, day trips, and date ideas,
receive random recommendations on demand, and rate and record their
experiences for posterity and to revisit in the future.

FEATURES:
- Random recommendations
- Favorite recommendations
- Planned items
- Completed items
- Duplicate detection
- Ratings
- Google Sheets sync
- NFC launch support
- Mobile-friendly interface
